# 在Preview显示多个视图并命名

这里，我们使用 `Group` 来将多个组件的预览进行分组。不指定任何的预览选项（如ContentView），Xcode 会在目前模拟器上显示预览。对于 `TopBarMenu` 与 `BottomBarMenu`， 我们告诉 Xcode 在容器视图中预览布局，图19.5 可让你更加了解预览是什么样。
