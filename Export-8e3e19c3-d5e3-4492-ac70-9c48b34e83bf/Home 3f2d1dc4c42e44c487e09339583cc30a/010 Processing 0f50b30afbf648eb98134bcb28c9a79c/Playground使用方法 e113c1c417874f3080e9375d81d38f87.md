# 写入Markdown方法

- 如果是正常显示直接状态直接按照语法写
- 如果是Markdown的渲染模式
    - 按照Markdown语法正常书写
    - 在前面加上“//：”就能标注改行是markdown语法

<aside>
🤔 其实就是在Playground中，通过“//：”来标注该行是markdown内容，不是可以执行的代码行

</aside>
