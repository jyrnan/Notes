# 煎牛排

# 预备

- 尽量提前解冻
- 擦到很干
- 先撒胡椒再撒盐
- 拍一拍牛肉

# 猛火煎肉

- 热锅，足够热（手放上方感觉烫手）
- 不要用冷轧橄榄油
- **每面煎1分钟**，按压牛排，包括侧面

# Rest第一次

- **静置8分钟**（最好把牛排放在铁格子上，下方悬空，上方用金属纸盖）
- 同时关火把铁锅降温

# 慢煎至熟度

- 中/小火热锅
- 放入黄油，蒜粒，可以加入香料，味道渗入黄油
- **每面40秒，把黄油淋上去。2-3个cycle即可**
- 提前一个熟度出锅

# Rest第二次

- 静止**8-10分钟**（避免满盘血水）

# 切肉

- 利刀，垂直肌肉纤维来切
- 撒盐后入口
